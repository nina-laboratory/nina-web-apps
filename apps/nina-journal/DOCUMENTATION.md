# Summary
Nina Journal is a online journal application for the Nina Labs suite. It consolidades information from multiple sources and display an easy to ready summary of Nina Labs (such as deployments, releases etc) and my Personal Progress (Fitness progression).
# Architecture
- Base: Bun, NextJS, Tailwind, Biome and Playwright
- Hosting: Docker Container + Azure container App
- Installable as a PWA
